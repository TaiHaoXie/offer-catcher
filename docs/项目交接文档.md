## 二、快速启动指南

### 1. 配置 API Key

编辑 `backend/.env` 文件：

```bash
# 使用 Kimi API（请填入自己的密钥，不要提交真实密钥到仓库）
LLM_MODEL=kimi
KIMI_API_KEY=your_kimi_api_key_here
KIMI_API_BASE=https://api.moonshot.cn/v1

# 或使用豆包
LLM_MODEL=doubao
DOUBAO_API_KEY=your_key_here
DOUBAO_API_BASE=https://ark.cn-beijing.volces.com/api/v3
```

### 2. 一键启动

```bash
cd ~/Desktop/offer-catcher
./start.sh
```

访问地址：
- 前端：http://localhost:3000
- 后端：http://localhost:8888
- API文档：http://localhost:8888/docs

## 五、API 端点列表

| 端点 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 服务状态 |
| `/health` | GET | 健康检查 |

| `/api/v1/atoms` | GET/POST | 经历原子库 |

1. **LLM API Key**：需要在 `backend/.env` 中配置真实 Key，默认示例值不能完成模型调用。

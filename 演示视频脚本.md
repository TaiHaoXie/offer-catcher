## 核心功能演示（1:20 - 2:40）

### 功能一：智能匹配（1:20 - 1:50）

| 画面 | 音频/旁白 |
|------|-----------|
| 用户粘贴简历文本 | **旁白**：第一步，上传你的简历。 |
| JD 输入框自动填充示例岗位（AI产品经理） | 输入目标岗位的 JD... |
| 点击"开始匹配"，动画加载 | 系统会从硬技能、软技能、教育、经验、项目五个维度分析匹配度。 |
| 匹配结果面板展开，显示 68.5 分，各维度进度条动画 | **旁白**：68.5 分，这是一个中等偏上的匹配度。 |
| 下方展开"优化建议"卡片 | 系统会告诉你：缺少什么、如何补充、哪些关键词需要突出。 |

### 功能二：经历原子库（1:50 - 2:20）

| 画面 | 音频/旁白 |
|------|-----------|
| 切换到"经历原子库"标签 | **旁白**：这是我们的核心创新——经历原子库。 |
| 点击"从简历生成"，加载动画 | 系统会自动把你的简历"乐高化"，拆解成可重用的原子模块。 |
| 原子卡片瀑布流展开：工作经历、项目经历、技能、教育 | 每一段经历都是一个独立的"原子"，可以自由组合。 |
| 手动创建新原子：填写表单提交 | 你也可以手动添加新经历，不断丰富你的原子库。 |
| 【特效】原子像乐高积木一样飞入、重组，形成一份新简历 | **旁白**：面对不同的岗位，只需调取相关原子，就能快速组装出针对性极强的简历。 |

### 功能三：生成优化简历（2:20 - 2:40）

## 附录：录制检查清单

### 环境准备
- [ ] 后端服务已启动（`python -m uvicorn app.main:app --port 8000`）
- [ ] 浏览器已打开前端（`index.html`）
- [ ] 准备测试简历文本
- [ ] 准备测试 JD 文本

### 功能测试顺序
1. [ ] 简历上传/粘贴
2. [ ] JD 输入（使用示例岗位）
3. [ ] 匹配计算
4. [ ] 从简历生成原子
5. [ ] 手动创建原子
6. [ ] 生成优化简历
7. [ ] 添加投递记录
8. [ ] 更新投递状态
9. [ ] 查看历史记录

### 录制技巧
- 使用 2K/4K 分辨率，后期可裁剪
- 保持鼠标移动平滑
- 等待加载动画完成再点击
- 关闭浏览器通知弹窗
- 使用快捷键（Cmd+Tab 切换窗口）保持流畅
